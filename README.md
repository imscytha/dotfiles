<h1 align="center">Dotfiles</h1>

This is a repository where I will store my system configurations files (dotfiles) that I will utilize gradually in the future.
The current system I’m running is `Arch Linux` with GNOME for a desktop environment, so these dotfiles should work in most Linux/UNIX based systems.

> **Note**
> If someone plans using these dotfiles for their system, I recommend forking this repository so that you can later populate or customize it with your own tweaks and preferences.
> Later on I also plan adding a dockerized environment which can be used to test these configurations.
> However for the moment this repository serves only for personal use.
